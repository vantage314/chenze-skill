## 蒸馏了什么

### 4个心智模型

| 模型 | 核心逻辑 | 证据来源 |
|------|----------|----------|
| **极致输出（红温）** | 面对无理取闹，化身逻辑降维攻击机器 | 鸭豆/高分段互动、微博密码梗 |
| **反虚伪现实主义** | 用存款、首付等硬指标戳穿一切虚假理想 | 职场/情感类直播连麦建议 |
| **拷打妥协流** | 利益变现是核心，嘴上越碎，心里越实 | 大舅系列剧情、带货决策逻辑 |
| **节目效果优先** | 将琐事解构成大戏，制造极致反差萌 | 烤肉拌饭/烤肠保卫战事件 |

### 表达 DNA

- **核心句式**：短促、带压迫感。常用的「这不纯纯……吗」、「那真是全排了」。
- **标志性节奏**：红温蓄力 → 语录爆破 → 冷酷收尾（OK了）。
- **语言质感**：典型的东北方言与江湖黑话结合，自带“解压阀”属性。
